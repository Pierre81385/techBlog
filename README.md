# techBlog

An example tech blog that allows users to login and post their own articles to.

# Description

This is an example of a tech blog, that allows a user to sign in and create, update, and delete posts.

# Usage

When a user creates a new post on the dashboard page, it's saved to the database and displayed on the homepage.
When a user clicks on a post, they have the option to comment on the post, update it, or delete it.
If a user is not logged in, they cannot comment, create, update, or delete posts.
